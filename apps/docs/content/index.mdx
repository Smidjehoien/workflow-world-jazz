# Workflow SDK

Build resilient, long-running applications with durable functions for JavaScript/TypeScript.

[Get Started](/docs/getting-started) · [Examples](/examples)

## What are workflows?

Workflows are reliable, multi-step processes that automatically resume after failures. Perfect for operations like order processing, user onboarding, AI agents, or any business logic that involves multiple API calls, delays, or external dependencies.

**The problem**: Traditional code is susceptible to sporadic failures, not durable, and bound by the max serverless function time.

**The solution**: Workflows automatically save progress after each step and resume exactly where they left off, no matter what interrupted them.

```ts
import { sleep } from '@vercel/workflow-core';

export async function processOrder(orderId: string) {
  "use workflow";
  
  // Each step is automatically retried and persisted
  const order = await fetchOrder(orderId);
  const payment = await processPayment(order.total);
  
  // Wait 1 day, then ship - the workflow suspends
  // and resumes where it left off
  await sleep("1d");
  await createShipment(order);
  
  await sendOrderConfirmation(order.email);
  
  return { orderId, status: "completed" };
}
```

## Features

**Automatic Retries** - Steps retry on failure with configurable delays and error types

**Sleep & Scheduling** - Built-in `sleep()` function for delays from seconds to years

**Webhooks** - Wait for external HTTP requests to resume workflow execution

**Durability Guarantees** - Progress persisted automatically, workflows resume after any interruption

**Parallel Execution** - Use `Promise.all()` and `Promise.race()` just like regular JavaScript

**Simple Async/Await** - Write workflows with familiar JavaScript patterns, no special syntax

**Built-in Fetch** - HTTP requests automatically converted to reliable, retryable steps

**Context & Metadata** - Access run IDs, timestamps, retry attempts, and other execution context

**Streaming Support** - Handle ReadableStreams and real-time data processing
