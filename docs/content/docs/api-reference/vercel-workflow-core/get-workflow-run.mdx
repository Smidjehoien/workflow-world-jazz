# `getWorkflowRun()`

Retrieves the workflow run metadata and status information for a given run ID. This function provides immediate access to workflow run details without waiting for completion, making it ideal for status checking and monitoring.

Use this function when you need to check workflow status, get timing information, or access workflow metadata without blocking on workflow completion.

## Import

```typescript
import { getWorkflowRun } from '@vercel/workflow-core/runtime';
```

- `runId` (string): Unique identifier for the workflow run
- `status` (string): Current status - 'pending', 'running', 'completed', 'failed', 'paused', or 'cancelled'
- `workflowName` (string): Name of the workflow function
- `deploymentId` (string): Deployment ID associated with the run
- `createdAt` (Date): When the workflow run was created
- `startedAt` (Date | null): When the workflow started executing (null if not started)
- `completedAt` (Date | null): When the workflow completed (null if not completed)
- `input` (any): The serialized input arguments passed to the workflow
- `output` (any | null): The workflow's return value (null if not completed)
- `error` (string | null): Error message if the workflow failed (null if successful)

## Good to Know

- Returns immediately without waiting for workflow completion
- Requires a valid `deploymentId` to be available in the environment
- The `input` and `output` fields contain serialized/hydrated values

## Examples

### Basic Status Check

Check the current status of a workflow run:

```typescript lineNumbers
import { getWorkflowRun } from '@vercel/workflow-core/runtime';

export async function GET(req: Request) {
  const url = new URL(req.url);
  const runId = url.searchParams.get('runId');

  if (!runId) {
    return Response.json({ error: 'No runId provided' }, { status: 400 });
  }

  try {
    const run = await getWorkflowRun(runId); // [!code highlight]

    return Response.json({
      runId: run.runId,
      status: run.status,
      workflowName: run.workflowName,
      createdAt: run.createdAt,
      startedAt: run.startedAt,
      completedAt: run.completedAt
    });
  } catch (error) {
    return Response.json(
      { error: 'Workflow run not found' },
      { status: 404 }
    );
  }
}
```
